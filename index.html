PLC Based Multistage Elevator Kit

A 3-floor elevator trainer kit developed using Siemens PLC and HMI through TIA Portal. The system implements full elevator logic including floor call handling, floor selection via HMI buttons, automatic door open/close sequences with delay timers, and comprehensive safety interlocks using Ladder Logic programming.

Key features include: Level-by-level movement control, Anti-collision safety logic, Real-time floor indication on HMI screen, Emergency stop functionality, and Overload protection simulation. This project demonstrated complete understanding of sequential PLC programming and HMI visualization for industrial elevator applications.

PLC + HMI

Smart Tank Monitoring System

Automated water tank level monitoring system using PLC and HMI, designed to replicate real-world industrial liquid management applications. The system uses level sensors to detect water levels and automatically controls pump operations for optimal performance.

Features include: Automatic pump start/stop based on tank level thresholds, Overflow prevention with high-level alarm, Dry-run pump protection with low-level cutoff, Live HMI screen showing animated tank levels, pump status, and alarm indicators. The project was programmed in TIA Portal and showcases practical application of PID-adjacent logic in fluid automation.

IoT Car Parking System

Smart parking management system developed using ESP8266 Wi-Fi module, IR proximity sensors, and servo motors. The system detects vehicle entry/exit in real time and automatically allocates parking slots with physical barrier control.

Features include: IR-based vehicle detection at entry and exit gates, Automatic servo-driven barrier operation, Real-time slot availability tracking, Cloud connectivity via Wi-Fi for remote live monitoring, LED display indication for available/occupied slots, and mobile-accessible dashboard. This project bridges IoT with physical automation systems, showcasing practical smart city technology applications.
