PLC Based Multistage Elevator Kit

Siemens S7-1200TIA Portal Ladder LogicHMI Visualization Safety Interlocks

Objective: Design & developement of universal PLC-based elevator trainer kit with complete hardware fabrication and flexible I/O interfacing with safe and reliable control system using relay logic and fail-safe circuits

Tools Used: Siemens S7-1200 PLC, TIA Portal V17, HMI panel (Diascreen), Ladder Logic programming.

Working Logic: The system operates based on I/O commands where user inputs from floor and cabin switches are processed through the PLC to control lift and door outputs. Input signals are stored using memory bits, and based on sensor feedback from proximity sensors, the PLC executes directional movement (UP/DOWN) and stops the lift at the required floor. The outputs are driven through relay circuits, which control the lift motor and door mechanism.

The hardware circuit includes relay logic for signal interfacing and motor control, along with a fail-safe protection system using limit switches and interlocks. These circuits ensure that even if the PLC output fails or gives incorrect signals, the system remains safe by preventing over-travel and restricting operation during unsafe conditions such as door movement.

Reactor tank for Bactch process System

TIA PortalLevel Sensors Pump ControlHMI Alarms Auto Start/Stop

Objective: Automatic Batch Reactor Process tank operation.

Tools Used: Siemens S7 - 1200 PLC Trainer Kit, TIA Portal V20, Batch Reactor Tank Kit.

Working Logic:A Batch Process Reactor System is used to mix and heat two liquids in equal proportions and subsequently fill the processed liquid into vessels transported by a conveyor. The entire operation is automatic and continuous.

The system consists of three process tanks. The first two tanks store raw liquids, while the third tank functions as the reactor tank. Each tank is equipped with low-level and high-level sensors to monitor liquid levels.

Solenoid Valves SV1 and SV2, connected to the first and second process tanks respectively, are used to transfer liquids into the reactor tank. These valves open simultaneously to ensure equal-rate filling and automatically close when the high-level sensor of the reactor tank is activated.

After the reactor tank is filled, a preset time delay is applied during which the liquid is heated and mixed. Upon completion of the heating and mixing process, the processed liquid is discharged into vessels placed on a conveyor system. The conveyor movement and filling operation are synchronized to ensure accurate and efficient filling.

IoT Smart Parking System

ESP8266IR Sensors Servo MotorsCloud Monitoring Wi-Fi

Objective: Develop a smart city-ready parking management system with real-time slot detection, cloud connectivity, and automated gate control.

Tools Used: ESP8266 Wi-Fi module, IR proximity sensors, servo motors for barriers, LCD display, BLYNK cloud dashboard via Wi-Fi.

Working Logic: IR sensors detect vehicle presence at entry/exit gates and at each parking slot. On detection, the ESP8266 triggers the servo motor to raise the barrier. Slot availability is tracked in real time and transmitted to a BLYNK cloud dashboard via Wi-Fi. LCD display show occupied/free status at each slot. The system prevents entry when all slots are full.

Traffic Signal PLC System

Ladder LogicTimer Blocks Emergency OverrideHMI Control 4-Way Intersection

Objective: PLC-controlled traffic signal management for a 4-road intersection with pedestrain crossing and intelligent priority and emergency override capability.

Tools Used: ISPsoft (Delta PLC software), Ladder Logic, HMI Screen with manual override buttons.

Working Logic: Programmable green/yellow/red timing for all four lanes using TON timer blocks. Emergency vehicle detection override provides priority clearance with instant phase skip. HMI enables manual override during incidents. Fault-detection logic prevents conflicting green signals across opposing lanes.

This system also Hazard blinker option which is used during night times as the yellow signal flickers and For each cycle of 4 foads the pedestrain crossing is allowed for a preset time.

Generator Auto Start System

PLC Relay LogicAuto Transfer Switch Safety InterlocksHMI Alerts Auto Shutdown

Objective: Objective: Ensure uninterrupted power supply.

Initial Conditions

• Mains supply present

• Generator stopped

Abnormal Conditions

• Overspeed → trip

• Low oil pressure → shutdown

• Electrical over load → trip

• High Temperature → trip

Tools Used: Delta PLC Software (ISPsoft), relay logic, HMI Screen with alarm management and Fault Indication.

Working Logic: Normal Operation Sequence

1. Mains failure detected

2. Generator starts after delay

3. Generator reaches rated speed

4. Load transferred

5. Mains restored → cool down → generator stops.
